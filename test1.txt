1. 원하는 폴더에서 git init  명령어로 .git 생성하여 깃 저장소 생성
2. git config --global user.name "" / git config --global user.email ""    로 사용자 정보 설정 /  git config --global --list  로 확인
3. git  add  파일명  으로 변경 내역 추가
4. git status  명령어로 현재 작업내역 확인
5. git  commit -m "커밋 내용" 으로 변경내역 반영
6. git log 로 commit 내역 확인가능
7. git hub 에서 원격저장소 만들어주고
8. git remote add "github  복사 주소" 로 원격저장소와 로컬저장소 연결 (확인 : git remote -v )
9. git push origin master 로 원격 저장소에 반
